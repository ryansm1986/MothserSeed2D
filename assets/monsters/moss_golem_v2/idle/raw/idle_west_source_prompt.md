# idle_west Source Prompt

Use the moss golem reference images, especially the west-facing profile, as visual references. Create one horizontal raw source strip for a 2D game sprite idle animation.

Asset: the same moss golem monster, west-facing, large bulky stone body, heavy fists, moss mats over shoulders/back/head, vine bands, green glowing eye, green chest core when visible, tan-gray stone palette, dark crisp pixel-art outline, same proportions and source pixel-art rendering language. Keep the source art style and proportions; do not make chibi, super-deformed, cute mascot, plush, toy, sticker, or any alternate style.

Canvas: one horizontal strip only, exactly 5 distinct full-body west-facing poses, left to right. Background must be 100% solid flat #FF00FF magenta with no gradients. No borders, no grid lines, no text, no labels.

Frame motion: 1 neutral heavy grounded idle stance; 2 slow inhale with torso/shoulders subtly rising and moss lifting; 3 weight settles forward with head/near fist dipping; 4 weight rolls back, front arm relaxes upward and moss strands sway; 5 soft exhale returning close to frame 1 but not identical.

Critical consistency rules: same creature identity, same palette, same west-facing direction, same pixel scale and bounding-box family across all 5 poses. Leave large empty magenta gutters between the visible pose bounds; no body part or moss strand may touch another pose or cross the strip edge. No floating detached effects outside the main silhouette.
